Histograms provide a way to categorize metric measurements. When creating the metric, the expected range is defined via minimumExpectedValue and maximumExpectedValue. If not provide, micrometer will use default values. Micrometer now defines buckets in this range. If there a specific interesting points, they can be defined via sla. Useful for tracking (as the name suggests) tracking SLAs.
