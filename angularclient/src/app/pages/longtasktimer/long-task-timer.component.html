The long task timer is used when observing a long running task WHILE it is running.
Unlike the other metric timers you can see the progress, while the timer is running and not when the timer has stopped. This mostly makes sense for long running operations like batch jobs.
It can be very useful for alerting purposes as you know how long it takes before the process has finished,
