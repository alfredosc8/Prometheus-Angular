Gauge (pronounced [geid í]) is used for monitoring changing objects.
Use case could be observing a cache (its size).
Metrics are not collected all the time but only when the corresponding endpoint gets called.

TimeGauge

In addition to the "regular" gauge, measuring in numbers, there is also a special kind of gauge used when the monitored value is time.
One use case would be monitoring the drift between two consumers on the same stream.
Its usage is the same as for the regular gauge but you pass a timeunit and the time function to the builder instead of the measurement function.
